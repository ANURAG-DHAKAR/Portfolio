# 🌲 Kiran Shetty - 3D Portfolio Website

A stunning professional 3D portfolio website showcasing the work of UI/UX designer Kiran Shetty. Built with React Three Fiber for real 3D graphics, featuring an immersive forest environment and smooth scroll-based interactions.

## ✨ Features

### Modern 3D Design
- **Real 3D Graphics**: Built with Three.js and React Three Fiber
- **3D Forest Environment**: Realistic trees with multiple depth layers
- **Scroll-Based Camera Movement**: Smooth parallax effects as you scroll
- **Floating 3D Elements**: Interactive 3D plants and particles
- **Dynamic Lighting**: Multiple light sources creating depth and atmosphere
- **Fog and Atmospheric Effects**: Enhanced visual depth

### Professional Sections

#### 1. Hero Section
- Large bold typography with 3D depth effects
- Animated entrance
- Call-to-action buttons
- Statistics showcase (Projects, Experience, Clients)

#### 2. About Section
- Split layout with visual and content
- Animated cards showcasing key strengths
- Floating experience badge
- Download CV button

#### 3. Skills Section
- Animated skill progress bars with percentage
- Categorized by skill type
- Specialization cards highlighting expertise areas
- Smooth reveal animations

#### 4. Education Section
- Timeline layout with alternating sides
- Institution details with achievements
- Additional certifications grid
- Animated reveals on scroll

#### 5. Projects Section
- Grid layout with hover effects
- High-quality project images
- Click to view full case studies
- Modal with Problem → Research → Solution → Impact format
- 6 featured projects:
  - Beat the Bell Rush
  - SafeWalk
  - AssignMate
  - Colleevnto
  - CricAuction
  - Brand & Print Design

#### 6. Contact Section
- Contact information cards
- Working contact form
- Social media links
- Availability status indicator
- Footer with copyright

### UI/UX Features
- **Smooth Scrolling**: Seamless navigation between sections
- **Scroll Indicator**: Animated guide for first-time visitors
- **Modern Navigation**: Fixed header with smooth hover effects
- **Mobile Menu**: Responsive hamburger menu for mobile devices

## 🎨 Design System

### Color Palette
- **Background**: Pure black (#000000)
- **Primary Text**: White (#FFFFFF)
- **Secondary Text**: White with opacity (rgba(255,255,255,0.7))
- **Accent Green**: #4a9d5f, #2d7a4a, #1a4d2e
- **Borders**: White with low opacity

### Typography
- **Headings**: Bold, uppercase, tracking-tight
- **Body**: Regular weight, comfortable line-height
- **Labels**: Uppercase, increased letter-spacing
- **System Fonts**: -apple-system, Roboto, Segoe UI

### Spacing & Layout
- Consistent padding and margins
- Responsive grid systems
- Maximum content width: 1280px (7xl)
- Mobile-first approach

## 🛠️ Technical Stack

- **React** - UI library
- **Three.js** - 3D graphics engine
- **React Three Fiber** - React renderer for Three.js
- **React Three Drei** - Useful helpers for R3F
- **Motion (Framer Motion)** - Smooth animations
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Modern icon library
- **React Intersection Observer** - Scroll-based reveals
